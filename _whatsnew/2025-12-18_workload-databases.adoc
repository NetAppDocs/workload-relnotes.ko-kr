=== Active Directory 통합 기능 향상

Workload Factory for Databases는 고급 생성 옵션을 사용하여 Microsoft SQL Server를 배포할 때 세 가지 새로운 Active Directory(AD) 필드를 포함합니다. 이러한 개선 사항을 통해 Active Directory 가입 기본 설정을 지정하고 관리형 서비스 계정을 사용할 수 있습니다.

새로운 AD 필드는 다음과 같습니다.

* 선호하는 도메인 컨트롤러
* 선호하는 조직 단위 경로
* 대상 Active Directory 그룹
